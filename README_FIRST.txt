-- 2007 DST Critical Packages --

Included in this tree are all needed packages to bring your machine(s), in sync with the 2007 DST changes.

Within BASE, the latest tzdata packages are available for RHEL 2.1, 3, and 4.  Additionally, glibc and all
needed dependancies have been included for RHEL 2.1.  These are /not/ needed for RHEL 3 and 4.

Within APPLICATIONS, are packages with known DST issues that have been fixed.  These include:
    
        IBM Java
        BEA Java
        GCJ Java
        Evolution-Data-Server
        Evolution
        Red Hat Network Satellite IBM Java Update

Specifics on installation can be found within DSTFAQ2007_final.pdf, or at:

    https://www.redhat.com/support/resources/faqs/dst
    http://kbase.redhat.com/faq/FAQ_80_7909.shtm

For anything else, or any questions please see:

    http://www.redhat.com/support/service/GSS_phone.html
